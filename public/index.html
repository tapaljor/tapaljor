Anonymous Chat

Live Demo

Event-driven communication for real-time anonymous messaging.

This project is a real-time chat application that allows users to communicate anonymously. It is built with Socket.io to provide seamless, bi-directional communication between users and the server, enabling instantaneous message exchanges. The server is powered by Node.js, handling requests efficiently and managing user sessions.

Key Features:

  • Real-time Messaging: New users are broadcasted to all connected users in real-time without any delays, thanks to the event-driven nature of Socket.io. One-to-one chat feature without disturbance.
  • Anonymous Communication: Users can join chatrooms and send messages without the need for account creation or personal details, ensuring privacy and anonymity. No history of chat or user is stored.
  • Responsive Design: The front-end is built with HTML5 and CSS to ensure a clean, intuitive user interface that works across all devices, including desktops, tablets, and smartphones.
  • Lightweight and Fast: The combination of Node.js and JavaScript ensures that the application runs smoothly, providing a fast and responsive experience for users.
  • Scalable Architecture: The app is designed to handle multiple simultaneous connections, making it suitable for both small and large groups of users.
  • Real-time Updates: Users can see when others are typing and when new users join or leave the chatroom.

    Technologies Used

  • Socket.io
  • Node.js
  • JavaScript (Vanilla)
  • HTML5 & CSS
  • MongoDB
